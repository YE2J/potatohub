## 一、绝对估值方法

### 1.1 DCF（现金流折现）

**核心公式**：
```
企业价值 = Σ FCF_t / (1+WACC)^t + 终值 / (1+WACC)^n
股权价值 = 企业价值 - 净债务
每股价值 = 股权价值 / 总股本
```

**第一步：预测自由现金流（通常5年）**
```
FCFF = EBIT × (1-税率) + 折旧摊销 - 资本支出 - 营运资本增加
简化版：FCFF ≈ 经营活动现金流 - 资本支出
```

**第二步：计算 WACC**
```
WACC = E/(D+E) × Ke + D/(D+E) × Kd × (1-T)

Ke（股权成本）= Rf + β × (Rm - Rf)
  - Rf：10年期国债收益率（A股约2.5%）
  - β：行业平均或公司beta（1.0-1.5）
  - Rm-Rf：股权风险溢价（A股约5-7%）

Kd：债务成本（贷款利率，约4-5%）
T：所得税率（25%）
```

**A股各行业 WACC 参考范围**：

| 行业 | WACC范围 | 参考β |
|------|---------|------|
| 消费 | 8-10% | 0.8-1.0 |
| 科技 | 10-13% | 1.2-1.5 |
| 金融 | 7-9% | 1.0-1.2 |
| 周期 | 9-12% | 1.0-1.4 |
| 公用事业 | 6-8% | 0.5-0.8 |

**第三步：终值计算**
```
永续增长法：TV = FCF_n × (1+g) / (WACC - g)
  - g：永续增长率（通常2-3%，不应超过GDP增长）

退出倍数法：TV = EBITDA_n × EV/EBITDA倍数
  - 参考行业均值或历史中位数
```

**第四步：敏感性分析（必须执行）**

输出格式：
```
### DCF敏感性分析（每股价值，元）

| WACC \ g | 2.0% | 2.5% | 3.0% |
|----------|------|------|------|
| 9.0%     | 32.5 | 35.8 | 40.2 |
| 9.5%     | 28.3 | 30.8 | 34.0 |
| 10.0%    | 24.8 | 26.7 | 29.1 |
| 10.5%    | 22.0 | 23.5 | 25.3 |
| 11.0%    | 19.6 | 20.8 | 22.2 |
```

### 1.2 DDM（分红折现模型）

**适用**：高分红股票（银行、公用事业、成熟消费公司）

```
两阶段DDM：
P = Σ D_t / (1+Ke)^t + D_n × (1+g) / [(Ke-g) × (1+Ke)^n]

戈登模型（单阶段）：
P = D_1 / (Ke - g)
```

**适用性检查清单**：
- [ ] 连续分红超过3年
- [ ] 稳定的分红率（>30%）
- [ ] 盈利可预测性强
- [ ] 通常不适合高成长股（不分红/低分红）

### 1.3 SOTP（分部加总估值）

**适用**：多元化集团

```
集团价值 = Σ 各业务分部估值 + 净现金 - 控股公司折价
```

示例框架：
| 分部 | 营收（亿） | 估值方法 | 估值（亿） |
|------|---------|---------|---------|
| 白酒 | 80 | 30x PE | 600 |
| 地产 | 50 | 0.6x PB | 120 |
| 金融 | 30 | 1.0x PB | 200 |
| **合计** | | | **920** |
| 控股折价 | | -15% | -138 |
| **集团估值** | | | **782** |

## 二、相对估值方法

### 2.1 PE Band（市盈率通道）

```
历史PE分位数分析：
- 取近5年PE_TTM时间序列
- 计算10%/25%/50%/75%/90%分位数
- 根据当前PE所在分位判断高估/低估
```

| 分位数 | PE | 隐含股价 | 解读 |
|------|-----|---------|------|
| 90% | 35x | 52.5 | 严重高估 |
| 75% | 28x | 42.0 | 偏贵 |
| 50% | 22x | 33.0 | 合理 |
| 25% | 16x | 24.0 | 便宜 |
| 10% | 12x | 18.0 | 严重低估 |
| **当前** | **18x** | **27.0** | **低估（30%分位）** |

### 2.2 PB-ROE 矩阵

```
理论关系：PB = (ROE - g) / (Ke - g)
实际使用：将同行业公司绘制在PB vs ROE散点图上
```

| 象限 | PB | ROE | 解读 |
|------|-----|-----|------|
| 右下 | 低PB | 高ROE | **低估（最佳买入区）** |
| 右上 | 高PB | 高ROE | 合理（优质溢价） |
| 左下 | 低PB | 低ROE | 价值陷阱或困境反转 |
| 左上 | 高PB | 低ROE | 高估（回避） |

### 2.3 EV/EBITDA

```
EV = 市值 + 净债务（有息负债 - 现金）
EBITDA = 营业利润 + 折旧 + 摊销

优势：
- 消除资本结构差异（vs PE）
- 消除折旧政策差异
- 适合重资产行业（电信/能源/基建）
```

**A股各行业 EV/EBITDA 参考**：

| 行业 | 中位数 | 低估 | 高估 |
|------|--------|------|------|
| 消费 | 15-20x | <12x | >25x |
| 科技 | 12-18x | <10x | >22x |
| 能源 | 6-10x | <5x | >12x |
| 公用事业 | 8-12x | <6x | >15x |

## 三、估值陷阱检测（10项必检）

| # | 陷阱 | 检测方法 | 典型案例 |
|---|------|---------|---------|
| 1 | 低PE周期顶部 | 盈利最高时PE最低，即将下降 | 2021年煤炭5xPE即顶部 |
| 2 | 高PE成长可被增长消化 | PEG<1说明增速支撑估值 | 30xPE+40%增速=PEG 0.75 |
| 3 | 低PB价值毁灭 | ROE持续<Ke意味着毁灭股东价值 | 长期亏损的重资产公司 |
| 4 | 商誉炸弹 | 商誉/净资产>30%存在减值风险 | 高溢价收购后业绩不达预期 |
| 5 | 应收款陷阱 | 应收/营收比率上升=收入质量差 | 政府应收+客户集中度高 |
| 6 | 资本化陷阱 | 研发/利息资本化美化利润 | 费用化后PE翻倍 |
| 7 | 一次性收益 | 扣非净利润与报表净利润差距大 | 卖资产/政府补贴增厚利润 |
| 8 | 股权稀释 | 股权激励/可转债摊薄EPS | PE应基于稀释后EPS |
| 9 | 关联交易 | 低价向关联方买/高价卖 | 利润转移至上市主体外 |
| 10 | 汇率波动 | 海外收入占比高=汇率敏感性大 | 人民币升值侵蚀出口商利润 |

## 四、估值方法选择决策树

```
公司类型是什么？
├── 成熟稳定（消费/公用事业/银行）
│   ├── 高分红 → DDM
│   └── 低分红 → DCF + PE Band
├── 高成长（科技/医药/新能源）
│   └── DCF（高增长期）+ PEG + PS
├── 周期股（煤炭/钢铁/有色）
│   └── PB + EV/EBITDA（避免PE）
├── 多元化集团
│   └── SOTP
└── 亏损公司
    └── PS（市销率）+ EV/Sales
```

## 五、交叉验证原则

**至少使用2种估值方法，取中值：**
1. DCF → 内在价值
2. 可比PE → 市场定价
3. 若差异>30% → 检查假设是否合理

## 注意事项

1. **DCF对假设高度敏感**：WACC变动1%可使估值变动20%+，敏感性分析是必须的
2. **可比公司必须真正可比**：同行业+同规模+同阶段，不要把龙头PE倍数套在小公司上
3. **A股估值体系独特**：壳价值/流动性溢价/政策溢价意味着不能直接套用美股标准
4. **估值不是目标价**：市场可以长期非理性，估值是锚而不是交易信号
5. **周期股特殊处理**：使用归一化盈利（周期中值盈利），而非当前盈利
6. **不适用于加密货币**：BTC/ETH等不适用传统估值框架

## 实际工作流

1. **获取数据** → Tushare MCP（优先）或 web_extract（回退）
2. **确定方法** → 根据决策树选择2-3种估值方法

5. **陷阱检测** → 逐项检查10项估值陷阱
6. **输出结果** → 按标准格式输出估值报告
