# 规则引擎策略设计 v3

## 五层判决架构

```
         ┌─────────────────────────────────┐
         │  ⑤ 风控层                       │
         │  止损7%/止盈20%/移动止损12%      │
         │  最长20天持仓                    │
         ├─────────────────────────────────┤
         │  ④ 趋势逆转层                    │
         │  GS方向突变: 多→空强卖, 空→多强买│
         │  (优先级高于综合打分)             │
         ├─────────────────────────────────┤
         │  ③ 冷静期层                      │
         │  卖出后3天禁止买入               │
         │  同向信号间隔2天                 │
         ├─────────────────────────────────┤
         │  ② 综合打分(10分制)              │
         │  买入分 vs 卖出分取高者           │
         │  strong≥7, medium≥5, weak≥3     │
         ├─────────────────────────────────┤
         │  ① 五维信号解析                  │
         │  GS趋势+暗盘资金+主力雷达         │
         │  +AI活跃度+主力持仓               │
         └─────────────────────────────────┘
```

## 打分规则

### 买入分（满分10分）

| 来源 | 条件 | 加分 |
|------|------|------|
| GS趋势 | 多头趋势 (bull_market或tcy) | +3 |
| GS信号 | G点触发 (kk0) | +3 |
| GS强势 | tcy (强势多头) | +1 |
| 暗盘资金 | 1日+3日连续流入 | +2 |
| 主力雷达 | buy_icon 或 buy_bottom | +2 |
| AI活跃度 | ≥3 (强势线) | +1 |
| AI活跃度 | ≥6 (大牛线) | +1 |
| 主力持仓 | >60且增仓>1% | +1 |

### 卖出分（满分10分）

| 来源 | 条件 | 加分 |
|------|------|------|
| GS趋势 | 空头趋势 (tkc) | +3 |
| GS信号 | S点触发 (pp0) | +3 |
| GS弱势 | tkc (弱势空头) | +1 |
| 暗盘资金 | 1日+3日连续流出 | +2 |
| 主力雷达 | sell_top | +2 |
| 主力持仓 | <40且减仓>1% | +1 |

### Conviction → 仓位

| Conviction | 总分门槛 | 目标仓位 |
|-----------|---------|---------|
| strong_buy | ≥7 | 100% |
| medium_buy | ≥5 | 50% |
| weak_buy | ≥3 | 25% |
| neutral | <3且<3 | 0% |
| weak_sell | ≥3卖出 | -25% |
| medium_sell | ≥5卖出 | -50% |
| strong_sell | ≥7卖出 | -100% |

## 判决优先级

1. **风控**: 如果已持仓, 先检查止损/止盈/移动止损/超时
2. **冷静期**: 卖出后N天锁仓, 只允许卖出不允许买入
3. **趋势逆转**: GS方向突变(prev_dir=1→gs_dir=-1)立即强卖
4. **综合打分**: 计算买入分和卖出分, 取高者
5. **持仓保持**: 无信号时已持仓的继续持有

## 已知问题

1. **持仓倾向1天** — 日频信号每日重判，容易隔天翻转。建议加"买入前3天不卖"的惯性规则
2. **华润三九亏损** — 该股波动小(防御股)，不适合短线频繁交易策略
3. **单票全仓** — 回撤大(40-57%)，组合层面风控可显著改善
4. **无LV2数据时信号弱** — 暗盘资金和主力持仓的近似值精度有限
